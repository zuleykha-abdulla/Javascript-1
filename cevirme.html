Əgər, daxil edilmiş məlumatların yazı şəklində yox, ədəd şəklində toplanmasını istəyiriksə bunun üçün dəyişənin tipini yazıdan ədədə çevirməliyik. Bunun ən asan yollarından biri dəyişəni 1-ə vurmaqdır. Nümunəyə baxaq:

var a=prompt("Birinci eded");
var b=prompt("Ikinci eded");
alert(a*1+b*1);

Dəyişənin tipini yazıdan ədədə çevirmək üçün başqa yollar da var. Bu yollardan biri parseInt() funksiyasından istifadə etməkdir. Nümunəyə baxaq:

var a=prompt("Birinci eded");
var b=prompt("Ikinci eded");
var c=parseInt(a);
var d=parseInt(b);
alert(c+d);

Bu funksiyanı istifadə etmək üçün mütləq stringlərə ehtiyac yoxdur. Bu funksiyanı adi ədədlərin də tam hissəsini tapmaq üçün istifadə etmək olar. Nümunəyə baxaq:

var a=parseInt(5.5);
var b=parseInt(6);
alert(a+b);
nəticə 11 olacaq

Əgər, ədədlərin tam hissələrini yox, özünü istifadə etmək istəyiriksə o zaman parseFloat() funksiyasından istifadə edə bilərik. Nümunəyə baxaq:

var a=prompt("Birinci eded");
var b=prompt("Ikinci eded");
var c=parseFloat(a);
var d=parseFloat(b);
alert(c+d);
nəticə 11,5 olacaq

Ədədlər və yazılardan əlavə məntiqi tip də mövcuddur. Məntiqi tip true və false qiymətlərini alır. True doğru, false isə yalnış mənasına gəlməkdədir. Bir şey doğru olduqda onun qiyməti true, yalnış olduqda isə false olur. Nümunəyə baxaq.

var a=true; // Dogru
var b=false; // Yalnis
alert(a);
alert(b);
